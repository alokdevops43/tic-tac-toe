# 🎮 Animated Tic Tac Toe

A modern, responsive, and animated **Tic Tac Toe** game built using **HTML, CSS, and Vanilla JavaScript**. This project features a beautiful glassmorphism interface, smooth animations, confetti celebrations, persistent score tracking, and an engaging user experience.

## ✨ Features

- 🎨 Modern Glassmorphism UI
- 🎮 Interactive Tic Tac Toe Gameplay
- 🏆 Winner Detection
- 🤝 Draw Detection
- 🎉 Confetti Celebration
- 🌈 Dynamic Win & Draw Themes
- 💫 Smooth Cell Animations
- 📊 Live Scoreboard
- 💾 Score Persistence with Local Storage
- 🔄 Restart Game Anytime
- ⚡ Ripple Button Effect
- 📱 Fully Responsive Design
- ⌨️ Keyboard Shortcut (Press **R** to Restart)
